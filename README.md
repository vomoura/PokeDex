# Pokédex

Uma Pokédex moderna e interativa construída com React e a PokéAPI.

## Características

- 🎨 Design moderno e responsivo
- 🔍 Busca de Pokémon em tempo real
- 📊 Visualização detalhada de stats
- 🎯 Interface intuitiva
- ⚡ Performance otimizada com Vite

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tecnologias

- React 18
- Vite
- PokéAPI
- CSS3
